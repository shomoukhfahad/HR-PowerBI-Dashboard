# HR Power BI Dashboard

An interactive **Power BI dashboard** providing comprehensive Human Resources analytics, focusing on workforce distribution, salary insights, and leave management trends.

> ⚠️ Note: All data displayed is anonymized and used for demonstration purposes only.

---

## 🎯 Project Objective

This dashboard was designed to support executive and HR decision-making by delivering:

- A consolidated workforce overview
- Salary distribution and compensation insights
- Employee demographic analysis
- Leave behavior trends
- Saudization (Localization) tracking against target

---

## 🛠 Tools & Technologies

- Power BI
- Power Query (ETL & Data Transformation)
- DAX (Measures & KPIs)
- Data Modeling (Star Schema Design)
